#  Student Portal System

A web-based Student Portal System developed using Django and HTML templates.  
It allows efficient management of student records with search functionality and full CRUD operations.

---

##  Features

- Student registration and login system  
- Add new student records  
- View all student records  
- Update existing student data  
- Delete student records  
-  Student search functionality (search by name / ID)  
- Admin dashboard for data management  
- Template-based frontend (HTML, CSS, Bootstrap)  
- Database integration using Django ORM  
- Simple and user-friendly interface  

---

##  Tech Stack

- Backend: Django (Python)  
- Frontend: HTML, CSS, Bootstrap (Templates)  
- Database: SQLite (default Django database)  
- Version Control: Git & GitHub
